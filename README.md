# Blocnote
A simple application for mobile which is a Bloc Note.

-----------------------------------------------------

- Objectives
  - Save texts / notes
  - Add to the favorite
  - Manage a note

- Action
  - Database
    - Id
    - Text
    - Favorite
    - Date
  
  - First layout
      - List of notes (date and favorite)
      - Access notes
      - Create a note
      - Add a note to favorite
      - Delete a note
      
  - Bloc note layout
      - Text
      - Possibility of a list
      - Add to the favorite
