# Blocnote
A simple application for mobile which is a Bloc Note.

-----------------------------------------------------

- Objectives
  - Save texts / notes
  - System of search for notes
  - Manage a note

- Action
  - Database
    - Id
    - Title (optional)
    - Texts
    - Favorite
    - Date
  
  - First layout
      - List of notes (date and favorite)
      - Access notes
      - Create a note
      - Add a note to favorite
      - Delete a note
      
  - Bloc note layout
      - Title (optional)
      - Text
      - Possibility of a list
